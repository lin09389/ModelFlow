## 📋 数据表详细设计

### 1. users - 用户表

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | BIGINT | 用户 ID | PRIMARY KEY, AUTO_INCREMENT |
| openid | VARCHAR(64) | 微信 openid | UNIQUE, INDEX |
| phone | VARCHAR(20) | 手机号 | UNIQUE |
| nickname | VARCHAR(50) | 昵称 | NOT NULL |
| avatar | VARCHAR(255) | 头像 URL | |
| role | TINYINT | 角色：1=用户，2=骑手，3=管理员 | DEFAULT 1 |
| student_id | VARCHAR(20) | 学号（骑手认证用） | |
| student_name | VARCHAR(50) | 真实姓名（骑手认证用） | |
| student_verified | TINYINT | 学生认证：0=未认证，1=已认证 | DEFAULT 0 |
| rider_verified | TINYINT | 骑手认证：0=未认证，1=已认证 | DEFAULT 0 |
| deposit | DECIMAL(10,2) | 保证金（骑手） | DEFAULT 0.00 |
| balance | DECIMAL(10,2) | 余额（骑手收入） | DEFAULT 0.00 |
| rating | DECIMAL(3,2) | 评分（骑手） | DEFAULT 5.00 |
| total_orders | INT | 完成订单数（骑手） | DEFAULT 0 |
| status | TINYINT | 状态：0=禁用，1=正常 | DEFAULT 1 |
| created_at | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |
| updated_at | DATETIME | 更新时间 | ON UPDATE CURRENT_TIMESTAMP |

**索引**:
- PRIMARY KEY (id)
- UNIQUE KEY uk_openid (openid)
- UNIQUE KEY uk_phone (phone)
- INDEX idx_role (role)
- INDEX idx_rider_verified (rider_verified)

---

### 2. orders - 订单表

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | BIGINT | 订单 ID | PRIMARY KEY, AUTO_INCREMENT |
| order_no | VARCHAR(32) | 订单编号 | UNIQUE |
| user_id | BIGINT | 下单用户 ID | FOREIGN KEY → users.id |
| rider_id | BIGINT | 接单骑手 ID | FOREIGN KEY → users.id, NULL |
| type | TINYINT | 订单类型：1=取物，2=送餐，3=代购，4=其他 | |
| title | VARCHAR(100) | 订单标题 | NOT NULL |
| description | TEXT | 订单详情 | |
| pickup_address | VARCHAR(255) | 取货地址 | |
| pickup_phone | VARCHAR(20) | 取货联系人电话 | |
| delivery_address | VARCHAR(255) | 送货地址 | NOT NULL |
| delivery_phone | VARCHAR(20) | 收货人电话 | NOT NULL |
| expected_time | DATETIME | 期望送达时间 | |
| reward | DECIMAL(10,2) | 悬赏金额 | NOT NULL |
| platform_fee | DECIMAL(10,2) | 平台服务费 | DEFAULT 0.00 |
| status | TINYINT | 状态：见下方状态说明 | DEFAULT 0 |
| weight | VARCHAR(20) | 物品重量/大小 | |
| images | JSON | 订单图片 | |
| rider_remark | TEXT | 骑手备注 | |
| user_remark | TEXT | 用户备注 | |
| accepted_at | DATETIME | 接单时间 | |
| picked_at | DATETIME | 取货时间 | |
| completed_at | DATETIME | 完成时间 | |
| cancelled_at | DATETIME | 取消时间 | |
| cancel_reason | VARCHAR(255) | 取消原因 | |
| created_at | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |
| updated_at | DATETIME | 更新时间 | ON UPDATE CURRENT_TIMESTAMP |

**索引**:
- PRIMARY KEY (id)
- UNIQUE KEY uk_order_no (order_no)
- INDEX idx_user_id (user_id)
- INDEX idx_rider_id (rider_id)
- INDEX idx_status (status)
- INDEX idx_created_at (created_at)

**订单状态**:
```
0 = 待接单（用户已下单，等待骑手接单）
1 = 已接单（骑手已接单，前往取货）
2 = 已取货（骑手已取货，配送中）
3 = 已完成（已送达，待确认）
4 = 已确认（用户确认收货，订单完成）
5 = 已取消（用户取消）
6 = 已退款（退款完成）
-1 = 异常/纠纷
```

### 3. payments - 支付记录表

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | BIGINT | 支付 ID | PRIMARY KEY, AUTO_INCREMENT |
| payment_no | VARCHAR(32) | 支付流水号 | UNIQUE |
| order_id | BIGINT | 关联订单 ID | FOREIGN KEY → orders.id |
| user_id | BIGINT | 支付用户 ID | FOREIGN KEY → users.id |
| amount | DECIMAL(10,2) | 支付金额 | NOT NULL |
| type | TINYINT | 类型：1=订单支付，2=保证金，3=充值 | |
| channel | TINYINT | 渠道：1=微信支付，2=支付宝 | |
| status | TINYINT | 状态：0=待支付，1=已支付，2=退款中，3=已退款 | DEFAULT 0 |
| transaction_id | VARCHAR(64) | 第三方交易号 | |
| paid_at | DATETIME | 支付成功时间 | |
| refund_at | DATETIME | 退款时间 | |
| created_at | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |

**索引**:
- PRIMARY KEY (id)
- UNIQUE KEY uk_payment_no (payment_no)
- INDEX idx_order_id (order_id)
- INDEX idx_user_id (user_id)
- INDEX idx_status (status)

---

### 4. reviews - 评价表

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | BIGINT | 评价 ID | PRIMARY KEY, AUTO_INCREMENT |
| order_id | BIGINT | 订单 ID | UNIQUE, FOREIGN KEY |
| user_id | BIGINT | 评价用户 ID | |
| rider_id | BIGINT | 被评价骑手 ID | |
| rating | TINYINT | 评分：1-5 星 | NOT NULL |
| content | TEXT | 评价内容 | |
| images | JSON | 评价图片 | |
| reply | TEXT | 骑手回复 | |
| created_at | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |

**索引**:
- PRIMARY KEY (id)
- UNIQUE KEY uk_order_id (order_id)
- INDEX idx_rider_id (rider_id)

---

### 5. withdrawals - 提现记录表（骑手）

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | BIGINT | 提现 ID | PRIMARY KEY, AUTO_INCREMENT |
| withdrawal_no | VARCHAR(32) | 提现流水号 | UNIQUE |
| rider_id | BIGINT | 骑手 ID | FOREIGN KEY → users.id |
| amount | DECIMAL(10,2) | 提现金额 | NOT NULL |
| fee | DECIMAL(10,2) | 手续费 | DEFAULT 0.00 |
| actual_amount | DECIMAL(10,2) | 实际到账金额 | |
| channel | TINYINT | 渠道：1=微信，2=支付宝 | |
| account | VARCHAR(100) | 提现账号（微信 openid/支付宝账号） | |
| real_name | VARCHAR(50) | 真实姓名 | |
| status | TINYINT | 状态：0=待审核，1=处理中，2=已到账，3=已拒绝 | DEFAULT 0 |
| remark | VARCHAR(255) | 驳回原因/备注 | |
| processed_at | DATETIME | 处理时间 | |
| created_at | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |

**索引**:
- PRIMARY KEY (id)
- UNIQUE KEY uk_withdrawal_no (withdrawal_no)
- INDEX idx_rider_id (rider_id)
- INDEX idx_status (status)

---

### 6. admin_logs - 后台操作日志表

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | BIGINT | 日志 ID | PRIMARY KEY, AUTO_INCREMENT |
| admin_id | BIGINT | 管理员 ID | |
| action | VARCHAR(50) | 操作类型 | |
| target_type | VARCHAR(20) | 目标类型：user/order/withdrawal | |
| target_id | BIGINT | 目标 ID | |
| content | TEXT | 操作详情 | |
| ip | VARCHAR(50) | 操作 IP | |
| created_at | DATETIME | 创建时间 | DEFAULT CURRENT_TIMESTAMP |

**索引**:
- PRIMARY KEY (id)
- INDEX idx_admin_id (admin_id)
- INDEX idx_created_at (created_at)

---

### 7. system_configs - 系统配置表

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | INT | 配置 ID | PRIMARY KEY, AUTO_INCREMENT |
| config_key | VARCHAR(50) | 配置键 | UNIQUE |
| config_value | TEXT | 配置值 | |
| description | VARCHAR(255) | 配置说明 | |
| updated_at | DATETIME | 更新时间 | ON UPDATE CURRENT_TIMESTAMP |

**默认配置**:
```
platform_fee_rate = 0.05        # 平台服务费率 5%
min_reward = 3.00               # 最低悬赏金额
max_reward = 500.00             # 最高悬赏金额
min_withdrawal = 10.00          # 最低提现金额
withdrawal_fee_rate = 0.01      # 提现手续费率 1%
rider_deposit = 50.00           # 骑手保证金
```

## 🔧 SQL 初始化脚本

```sql
-- 创建数据库
CREATE DATABASE IF NOT EXISTS campus_erao DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

USE campus_erao;

-- 创建 users 表
CREATE TABLE users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    openid VARCHAR(64) UNIQUE,
    phone VARCHAR(20) UNIQUE,
    nickname VARCHAR(50) NOT NULL,
    avatar VARCHAR(255),
    role TINYINT DEFAULT 1,
    student_id VARCHAR(20),
    student_name VARCHAR(50),
    student_verified TINYINT DEFAULT 0,
    rider_verified TINYINT DEFAULT 0,
    deposit DECIMAL(10,2) DEFAULT 0.00,
    balance DECIMAL(10,2) DEFAULT 0.00,
    rating DECIMAL(3,2) DEFAULT 5.00,
    total_orders INT DEFAULT 0,
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_role (role),
    INDEX idx_rider_verified (rider_verified)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 创建 orders 表
CREATE TABLE orders (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    order_no VARCHAR(32) UNIQUE,
    user_id BIGINT NOT NULL,
    rider_id BIGINT,
    type TINYINT NOT NULL,
    title VARCHAR(100) NOT NULL,
    description TEXT,
    pickup_address VARCHAR(255),
    pickup_phone VARCHAR(20),
    delivery_address VARCHAR(255) NOT NULL,
    delivery_phone VARCHAR(20) NOT NULL,
    expected_time DATETIME,
    reward DECIMAL(10,2) NOT NULL,
    platform_fee DECIMAL(10,2) DEFAULT 0.00,
    status TINYINT DEFAULT 0,
    weight VARCHAR(20),
    images JSON,
    rider_remark TEXT,
    user_remark TEXT,
    accepted_at DATETIME,
    picked_at DATETIME,
    completed_at DATETIME,
    cancelled_at DATETIME,
    cancel_reason VARCHAR(255),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (rider_id) REFERENCES users(id),
    INDEX idx_user_id (user_id),
    INDEX idx_rider_id (rider_id),
    INDEX idx_status (status)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 创建 payments 表
CREATE TABLE payments (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    payment_no VARCHAR(32) UNIQUE,
    order_id BIGINT,
    user_id BIGINT NOT NULL,
    amount DECIMAL(10,2) NOT NULL,
    type TINYINT NOT NULL,
    channel TINYINT,
    status TINYINT DEFAULT 0,
    transaction_id VARCHAR(64),
    paid_at DATETIME,
    refund_at DATETIME,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (order_id) REFERENCES orders(id),
    FOREIGN KEY (user_id) REFERENCES users(id),
    INDEX idx_order_id (order_id),
    INDEX idx_user_id (user_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 创建 reviews 表
CREATE TABLE reviews (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    order_id BIGINT UNIQUE,
    user_id BIGINT NOT NULL,
    rider_id BIGINT NOT NULL,
    rating TINYINT NOT NULL,
    content TEXT,
    images JSON,
    reply TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (order_id) REFERENCES orders(id),
    FOREIGN KEY (rider_id) REFERENCES users(id),
    INDEX idx_rider_id (rider_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 创建 withdrawals 表
CREATE TABLE withdrawals (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    withdrawal_no VARCHAR(32) UNIQUE,
    rider_id BIGINT NOT NULL,
    amount DECIMAL(10,2) NOT NULL,
    fee DECIMAL(10,2) DEFAULT 0.00,
    actual_amount DECIMAL(10,2),
    channel TINYINT,
    account VARCHAR(100),
    real_name VARCHAR(50),
    status TINYINT DEFAULT 0,
    remark VARCHAR(255),
    processed_at DATETIME,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (rider_id) REFERENCES users(id),
    INDEX idx_rider_id (rider_id),
    INDEX idx_status (status)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 创建 admin_logs 表
CREATE TABLE admin_logs (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    admin_id BIGINT,
    action VARCHAR(50),
    target_type VARCHAR(20),
    target_id BIGINT,
    content TEXT,
    ip VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_admin_id (admin_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 创建 system_configs 表
CREATE TABLE system_configs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    config_key VARCHAR(50) UNIQUE,
    config_value TEXT,
    description VARCHAR(255),
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 插入默认配置
INSERT INTO system_configs (config_key, config_value, description) VALUES
('platform_fee_rate', '0.05', '平台服务费率 5%'),
('min_reward', '3.00', '最低悬赏金额'),
('max_reward', '500.00', '最高悬赏金额'),
('min_withdrawal', '10.00', '最低提现金额'),
('withdrawal_fee_rate', '0.01', '提现手续费率 1%'),
('rider_deposit', '50.00', '骑手保证金');

-- 创建默认管理员账号
INSERT INTO users (openid, phone, nickname, role, rider_verified, status) 
VALUES ('admin', '13800000000', '系统管理员', 3, 0, 1);
```
